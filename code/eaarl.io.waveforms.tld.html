Read records from file

Parameters
f : filehandle
Open readable filehandle
offset : int
Offset into file to start reading at
count : int
Number of records to read
progress : tqdm.tqdm or compatible or None or False
If provided, then progress.update() will be called after each raster is loaded. This is intended to support display of a progress bar via tqdm.
